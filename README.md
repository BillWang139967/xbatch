# xbatch

批量部署/运维工具

### 架构

```
     +------+  +------+  +------+
     |server|  |server|  |server|
     +------+  +------+  +------+
             \    |    /<-------------------------ssh
               +------+
               |xbatch|
               +------+
```
> * [使用手册](https://github.com/BillWang139967/xbatch/wiki)
> * [提 Bug](https://github.com/BillWang139967/xbatch/issues/new)
